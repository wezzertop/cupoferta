# 7. Roadmap y Próximas Mejoras

## Funcionalidades Pendientes

### Alta Prioridad

- [ ] **Búsqueda Avanzada**
  - Implementar con Algolia o Supabase Search
  - Filtros múltiples (precio, categoría, tienda)
  - Búsqueda por ubicación

- [ ] **Mejor Editor de Markdown**
  - Soporte nativo de emojis
  - Vista previa en tiempo real
  - Atajos de teclado

- [ ] **App Móvil (PWA)**
  - Progressive Web App
  - Installable en móviles
  - Notificaciones push

### Media Prioridad

- [ ] **Procesamiento de Imágenes en Servidor**
  - Edge Functions para optimizar
  - Compresión automática
  - Generación de thumbnails

- [ ] **Sistema de Gamificación**
  - Badges por logros
  - Ranking de usuarios
  - Ofertas destacadas del mes

- [ ] **Integración con Más Redes**
  - Twitter/X sharing automático
  - Facebook
  - WhatsApp

### Baja Prioridad

- [ ] **Modo Offline**
  - Service Workers
  - Cache de ofertas vistas

- [ ] **Analytics Dashboard**
  - Estadísticas de uso
  - Popularidad de ofertas
